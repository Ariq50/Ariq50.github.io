Homework 6 - Objects

Part 1

Page that contains a forms that takes user input in the form of a number with at least 4 decimals and returns the number rounded to the nearest integer, the square root of the number (rounded to the nearest integer), and the number rounded to the nearest tenth, hundredth, and thousandth position.

Part 2

Page that contains a form that takes user input in the form of long text content, followed by a single character. The long text content is then searched for the character and the number of instances the character appears in the content is returned in a text output box. If the character is not found in the long text content, a window opens with a message indicating this. The option is presented to the user to close said window.

Part 3

Page that contains a form that takes user input in the form of a ten digit phone number, in the format (999) 999-9999. The phone number is split into three sub-sections (area code, followed by the first three digits and the last four digits. Each of these sub-sections is then displayed in a text output box.
